# Krystal Discord Bot 💎

Krystal is a high-performance, event-driven Discord service built with **Node.js** and **Discord.js**. It features a modular command handler and asynchronous execution to ensure low-latency responses for community management.

---

## 🚀 Technical Architecture

* **⚡ Asynchronous Event Loop:** Leverages Node.js non-blocking I/O to handle concurrent WebSocket events from the Discord Gateway.
* **📂 Modular Command Handler:** Commands are dynamically loaded from the `/commands` directory, allowing for easy scaling without modifying the core engine.
* **🤖 Modern Discord.js (v14+):** Utilizes latest Gateway Intents and Slash Command integration for a modern user experience.
* **📦 Dependency Management:** Managed via **NPM**, ensuring consistent environments across development and production.

---

## 🛠️ Tech Stack
* **Runtime:** Node.js
* **Library:** Discord.js
* **Language:** JavaScript (ES6+)
* **Environment:** Dotenv (for secure token management)

---

## ⚙️ Installation & Deployment

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/MathewsGit/Krystal-Discord-Bot.git](https://github.com/MathewsGit/Krystal-Discord-Bot.git)
   ```
2. **Install Packages:**
```bash
npm install
```
3. **Configure Environment:**
Create a .env file:
```bash
TOKEN=your_bot_token_here
```

4. **Launch:**
```bash
node main.js
```
